라우터: 주소까지의 최적의 경로를 찾아줌, 라우팅 테이블을 사용
Amazon Virtual Private Colud(VPC)는 AWS에서 사용 가능한 프라이빗 가상 네트어크 공간
인터넷 게이트웨이: 서브넷 안에 있는 EC2와 같은 자원이 인터넷ㅎ과 통신 할 수 있게 함
인터넷 게이트웨이를 생성하고 서브넷의 라우팅 테이블에 설정하면 인터넷과 VPC가 통긴 가능
EC2의 경우 퍼블릭 IP와 Elastic IP가 있음, 퍼블릭IP는 재부팅시 변경

프라이빗 서브넷에 있는 자원은 일반적으로 인터넷과 통신 불가 하지만 필요한 경우 NAT 겡리트웨이 사용 가능
NAT은 Network Address Translation의 약자로 프라이빗 주소를 퍼브릭 주소로 변환
